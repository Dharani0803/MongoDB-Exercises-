# MongoDB-Exercises
# 📚 MongoDB CRUD – Library Database

## 📌 Objective
To practice MongoDB CRUD operations using a library/books collection.

---

## 🗂️ Database & Collection
- Database: `library`
- Collection: `books`

---

## 🔹 Operations Performed

### ✔ Create
- Inserted multiple books using `insertMany()`

### ✔ Read
- Retrieved all books  
- Filtered books by author  
- Found earliest and latest published books  

### ✔ Update
- Updated published year of a book  
- Added new field (`genre`)  

### ✔ Delete
- Deleted a specific book (e.g., "1984")  
- Removed books before a certain year  

---

## 🔹 Advanced Queries
- Top 3 recently published books  
- Search books using keywords ("MongoDB", "NoSQL")  

---

## 💡 Use Case
This simulates a **Library Management System** where books are stored, updated, and searched efficiently.

---

## 🎯 Outcome
- Gained hands-on experience with MongoDB  
- Learned advanced queries and filtering  
- Understood document-based data handling  

---

## 🏁 Conclusion
This task improved understanding of MongoDB operations and how they are used in real-world applications like library systems.
